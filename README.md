### –ó–∞–∫–ª—é—á–µ–Ω–∏–µ: –ò–≥—Ä–æ–≤—ã–µ –∞–≤—Ç–æ–º–∞—Ç—ã –æ–±–µ–∑—å—è–Ω–∫–∏ ‚Äî —ç—Ç–æ –æ—Ç–ª–∏—á–Ω–∞—è –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å –¥–ª—è –±–µ—Å–ø–ª–∞—Ç–Ω–æ–π –∏–≥—Ä—ã –∏ –≤—ã–∏–≥—Ä—ã—à–∞!

**–ò–≥—Ä–æ–≤—ã–µ –∞–≤—Ç–æ–º–∞—Ç—ã –æ–±–µ–∑—å—è–Ω–∫–∏** ‚Äî —ç—Ç–æ —É–≤–ª–µ–∫–∞—Ç–µ–ª—å–Ω—ã–π —Å–ª–æ—Ç, –∫–æ—Ç–æ—Ä—ã–π –¥–æ—Å—Ç—É–ø–µ–Ω –¥–ª—è –∏–≥—Ä—ã –±–µ—Å–ø–ª–∞—Ç–Ω–æ –≤ –¥–µ–º–æ-—Ä–µ–∂–∏–º–µ. –≠—Ç–æ –æ—Ç–ª–∏—á–Ω—ã–π —Å–ø–æ—Å–æ–± –ø–æ–ø—Ä–æ–±–æ–≤–∞—Ç—å —Å–≤–æ–∏ —Å–∏–ª—ã –±–µ–∑ —Ä–∏—Å–∫–∞ –ø–æ—Ç–µ—Ä–∏ –¥–µ–Ω–µ–≥, –∞ —Ç–∞–∫–∂–µ –Ω–∞—É—á–∏—Ç—å—Å—è –ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è –±–æ–Ω—É—Å–∞–º–∏ –∏ —Å—Ç—Ä–∞—Ç–µ–≥–∏—è–º–∏ –¥–ª—è —É—Å–ø–µ—à–Ω–æ–π –∏–≥—Ä—ã. –°–ª–æ—Ç—ã —Å –≤—ã—Å–æ–∫–∏–º RTP –∏ –±–æ–Ω—É—Å–Ω—ã–º–∏ —Ñ—É–Ω–∫—Ü–∏—è–º–∏ –ø—Ä–µ–¥–ª–∞–≥–∞—é—Ç –æ—Ç–ª–∏—á–Ω—ã–µ —à–∞–Ω—Å—ã –Ω–∞ –≤—ã–∏–≥—Ä—ã—à–∏ –∏ –¥–µ–ª–∞—é—Ç –∏–≥—Ä—É –µ—â–µ –±–æ–ª–µ–µ –∑–∞—Ö–≤–∞—Ç—ã–≤–∞—é—â–µ–π.
